desiresalesportal

Sales Portal for Desire Foundation

This is a portal to log and manage the sales data of Desire Foundation. Previous method had a lot of intermediaries and was prone to errors. This project aims to streamline the data management process.


Build Status

Tools Used
  • The database and authentication is handled by Firebase.
  • The UI is developed in React with Typescript.
  • Bootstrap is used for CSS components

live at https://desiresalesportal.team
